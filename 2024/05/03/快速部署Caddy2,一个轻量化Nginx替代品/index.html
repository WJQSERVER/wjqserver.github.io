快速部署Caddy2,一个轻量化Nginx替代品

发表于2024-05-03|更新于2024-05-03|建站
|阅读量:

Caddy2 是一款基于 Go 语言编写的强大且可扩展的平台，用于提供网站、服务和应用程序的服务。它是 Caddy 1 的升级版本，具有更多功能和改进

特点

  • 自动 HTTPS：Caddy2 内置了自动获取和管理 HTTPS 证书的功能，使您的网站可以安全地使用 HTTPS

  • 简单配置：Caddy2 提供了简单易用的配置文件（Caddyfile），使您可以轻松配置和管理您的网站

  • 插件系统：Caddy2 支持插件系统，可以通过添加插件来扩展其功能，例如日志记录、反向代理、缓存等

  • 高性能：Caddy2 使用 Go 语言编写，具有出色的性能和并发处理能力

优点

1.便于配置

Caddy2 的主要配置方式是使用 Caddyfile，它是一个文本文件，用于定义您的网站的配置和行为。使用Caddyflie最小化实现Hello World的配置如下。

Caddyfile 使用简单的语法，可以轻松地配置各种功能，如路由、TLS、代理等

此外，Caddy2 也支持使用 JSON 格式的原生配置语言进行配置

2.自动TLS

Caddy2默认开启HTTPS，支持自动化申请证书及续订，更适合小规模证书管理

3.支持更多先进技术

Caddy2默认使用HTTP2，同时也支持修改配置开启HTTP3和HTTP2 on TCP

缺点

1.插件安装麻烦

虽然目前已经引入了caddy add-package命令，但是插件冲突和构建不成功任然广泛存在

2.功能实现过于依赖插件

虽然Caddy2已比Caddy1大有改进，但是类似cache,brotli等常用功能任然需要插件实现
